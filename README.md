# parcel-starter
Parcel template with pug and sass


## 1. npm run dev

This command will start the app from the source files (/src).

```
http://localhost:3000/
```

## 2. Build

- Build the app:

```
$ npm run build
```

## 3. compress your images with tinify

- got to tinify.config.js and add tinify api key, then run script:

```
$ npm run tinify
```
